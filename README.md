# Portfolio

Modernized personal portfolio built with React, TypeScript, Vite, and Tailwind CSS.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## Vercel

This project is ready to deploy to Vercel as a Vite app.

## Tech stack

- React
- TypeScript
- Vite
- Tailwind CSS
- lucide-react
